1. 디렉토리 구조
	1) Assets
		사진, CSS, 폰트 등 저장
	
	2) Components
		버튼, 모달, 카드 등 사용자 UI 등 모듈 파일
		체크박스, Input 등 Form 파일 등의 컴포넌트 파일
	
	3) Context
		React Context 파일
	
	4) Data
		프로그램에서 사용되는 정보가 포함된 JSON 파일 같은 데이터를 저장
	
	5) Hooks
		프로젝트에서 사용하는 모든 커스텀 Hook
	
	6) Layouts
		네비게이션, 컨테이너, 사이드 등 레이아웃과 관련된 파일
	
	7) Pages
		각 페이지의 루트 파일과 해당 페이지에 속하는 모든 파일이 페이지별 폴더에 저장
			> 해당 페이지 전용 Hook 파일이 있다면 해당 페이지 디렉토리에 저장
	
	8) Services
		외부 API와 상호작용하는 모든 코드 파일
	
	9) Utils
		포맷팅 함수 등 모든 유틸리티 기능
